语法规则：

场景 允许的语法 禁止的语法 原因
全局变量 var x = []T{} var x []T{} 语法规则限制
var x []T (零值) x := []T{} := 不能在全局使用
局部变量 x := []T{} var x []T{} 局部声明不支持此语法
var x = []T{} x := []T 缺少 {}
函数参数/返回值 func f(a []T) func f(a []T{}) {} 只用于字面量
