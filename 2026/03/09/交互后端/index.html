参数 含义
CGO_ENABLED=0 禁用 CGO（C 语言绑定），确保编译出纯静态二进制文件，不依赖任何 .so 库
GOOS=linux 目标操作系统是 Linux（你的 Mac 是 darwin）
GOARCH=amd64 目标 CPU 架构是 64 位 x86
-o bin/openclaw_assistant_linux 输出到 bin/ 目录，文件名带 _linux 后缀便于区分
